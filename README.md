# shopProject

My project has 3 types of users.
* guest
1. viewing products
2. product search
3. registration
* user
1. authorization
2. user profile
3. product search
4. formation of cart
5. viewing products
6. adding a product on the website
7. can correct product descriptions
8. deleting own products
9. shopping cart management
10. create orders
* administrator
1. can correct product descriptions
2. delete products
3. ban users
4. cancel or accept orders
